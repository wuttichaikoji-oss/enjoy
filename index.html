Cloud Sync Firebase / Firestore

Version 3 ใช้ได้ 2 โหมด: Local เก็บข้อมูลในเครื่องเหมือนเดิม หรือ Cloud Sync เพื่อซิงก์ข้อมูลหลายเครื่องผ่าน Firebase ของคุณเอง โดยไม่มีหน้าล็อกอินให้ผู้ใช้เห็น

ต้องเปิด Cloud Firestore และ Anonymous Authentication ใน Firebase ก่อนใช้งาน Cloud Sync
ใช้รหัสเดียวกันทุกเครื่องเพื่อแชร์ข้อมูลชุดเดียวกัน
Local only
ยังไม่ได้เชื่อม Cloud
ตัวอย่าง Firestore Rules สำหรับทดลอง 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /rubberSitpin/{shopId}/{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
